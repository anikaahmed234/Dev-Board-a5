Question-1: What are the different ways to select an element in the DOM?

There are five methods to select an element in the DOM: getElementsByTagName(), getElementsByClassName(), getElementById(), querySelector(), and querySelectorAll(). In most cases, getElementById() is the preferred choice, though other methods have their uses as well. The simplest way to locate an HTML element in the DOM is by using its ID. If the element exists, the method returns it as an object. If it does not exist, the returned value will be null

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Key differences between innerHTML, innerText, and textContent:
innerHTML returns all text, including HTML tags, within an element.
innerText returns only the visible text inside an element and its child elements.
textContent retrieves all text within an element and its children, including hidden text, but without formatting.

Question-3: What is event delegation in the DOM?

Event delegation is a JavaScript technique where, instead of attaching individual event listeners to multiple child elements, a single listener is added to their parent.
This approach leverages event bubbling, where an event starts at the target element and propagates up the DOM tree.
It is particularly useful for handling a large number of dynamically created elements, improving performance by requiring only one event listener instead of multiple separate ones.

Question-4: What is event bubbling in the DOM?

Event bubbling occurs when an event originates from the clicked element, whether it's a child element or a container div, and then propagates upward through its parent, grandparent, and beyond until it reaches the tag of the HTML document.
Inner (child) elements execute their event listeners first, followed by outer (parent) elements as the event moves up the DOM hierarchy.

Question-5: How do you create, add, and remove elements using JavaScript?

In JavaScript, we can create elements, add them to the DOM, and remove them as needed.
1. Create an element using document.createElement("tagName"), where tagName represents HTML tags like <p>, <h1>, <div>, etc.
2. Add an element using the .appendChild() method, which inserts the new element inside a parent element.
3. Remove an element using .remove() for direct removal or .removeChild() to remove a specific child element from its parent.
